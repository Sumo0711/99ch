# 99圈圈叉叉  
網址：https://nine9ch.onrender.com/  
## 遊戲規則
1. 遊戲架構
大棋盤：這是一個 3x3 的大格局。  
小棋盤：大棋盤裡的每一格，裡面又包含了一個標準的 3x3 小井字遊戲。  
所以總共有9x9=81個格子。  
  
2. 核心規則（這也是最有趣的地方）  
這遊戲最特別的規則在於「你下的位置，決定了對手下一手必須在哪裡下」。  
  
舉例：  
假設我在「中間的小棋盤」裡，畫了一個 X 在右下角的格子。  
那麼，對手下一手必須去大棋盤右下角的小棋盤裡下 O。  
對手在那裡下的具體位置（例如他下了左上角），又會決定你下一手必須去大棋盤左上角的小棋盤裡下。  
  
3. 如何獲勝  
贏得小棋盤：在任何一個小棋盤中，如果你連成一線（直、橫、斜），你就「佔領」了那個大格子（該區視為一個巨大的 X 或 O）。  
  
贏得全場：當你在大棋盤上，將佔領的格子連成一線（直、橫、斜），你就贏得了整場遊戲。  
  
4. 特殊情況（自由權）  
如果不幸你被對手「送」到了一個已經分出勝負（或者已經被填滿平手）的小棋盤裡，你該怎麼辦？  
這時你獲得「自由權」：你可以選擇在棋盤上任意一個未完成的小棋盤中下子。
